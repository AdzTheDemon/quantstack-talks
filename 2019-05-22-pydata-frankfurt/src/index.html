What is Jupyter?

Jupyter is a consistent set of tools meant to facilitate the workflow of engineers, scientists

  • from the exploratory phase of their work

  • to the presentation of their result
  • including all the intermediary phase, HPC, visualization, debugging
  • making the experience actually FUN

The Jupyter Notebook

  • Interactive, browser-based computing environment

  • Exploratory data science, ML, visualization, analysis, stats
  • Reproducible document format:
    • Code
    • Narrative text (markdown)
    • Equations (LaTeX)
    • Images, visualizations
  • Over 50 programming languages
  • Everything open-source (BSD license)

Jupyter 2018 - 2019 update

JupyterLab

JupyterLab, the Project Jupyter next-generation web frontend and successor to the Jupyter Notebook.

  • JupyterLab is ready for users.
  • entirely new codebase 4 years in the making.
  • comes with many tools for a scientific workflow, such as:
    • Jupyter notebook with Jupyter widgets support.
    • console, text editor, terminal, and grid viewer.
  • We are working on real-time collaborative editing and stabilizing the extension API for JupyterLab core components.

Kernels

The Xeus C++ library is a standard implementation of the Jupyter kernel protocol

  • A Kernel for interpreted C++, xeus-cling
  • A new Python kernel, xeus-python
  • A new R kernel, JuniperKernel

JupyterHub & Binder

JupyterHub provides a platform for multi-user scientific computing

  • JupyterHub 1.0 released May 3rd 2019.
    • Named servers - configurable number of named servers, such as "research" and "teaching"
    • Improved spawn behavior
    • Total internal encryption
    • Authentication expiration and refresh - get new oauth tokens
  • Binder reaching 100k launches per week. BinderHub out of beta!
